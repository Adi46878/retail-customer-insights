#  Retail Customer Insights Analysis

##  Project Overview

This project focuses on analyzing retail transaction data to extract meaningful customer insights. The goal is to understand customer behavior, purchasing patterns, and key business metrics that can help improve decision-making.

---

##  Objectives

* Perform data cleaning and preprocessing
* Analyze customer purchasing behavior
* Identify top-performing products and categories
* Generate business insights using data visualization

---

##  Dataset

The dataset includes:

* **Customer.csv** → Customer information
* **Transactions.csv** → Purchase records
* **prod_cat_info.csv** → Product category details

---

## 🛠️ Tools & Technologies

* Python 
* Pandas
* NumPy
* Matplotlib / Seaborn
* Jupyter Notebook

---

## 🔍 Steps Performed

1. Data Cleaning (handling missing values, formatting)
2. Data Merging (combining multiple datasets)
3. Exploratory Data Analysis (EDA)
4. Visualization of trends and patterns
5. Insight generation

---

## 📊 Key Insights

* Identified top-selling product categories
* Analyzed customer purchase frequency
* Observed seasonal trends in transactions
* Derived actionable business recommendations
